## Статусная модель заявки

(Ссылка на изображение: *Рисунок – статусная модель заявки*)
[статусная-модель](./makets/статусная-модель.png)

**Статусы заявки:**
• Новый
• Назначен исполнитель
• Делегирована в другой отдел
• В работе
• Отклонена
• Завершен

**Таблица изменения статусов:**

| Текущий статус | Целевой статус | Описание |
|----------------|----------------|----------|
| - | Новый | Заявка создана, исполнитель будет назначен автоматически или вручную. |
| Новый | Назначен исполнитель | Для заявки назначен исполнитель. |
| Назначен исполнитель | Отклонена | Заявка отклонена. |
| Назначен исполнитель | В работе | Исполнитель приступает к работе. |
| Назначен исполнитель | Делегирована в другой отдел | Работы не относятся к текущему отделу. |
| Назначен исполнитель | Новый | Передача другому исполнителю (с признаком “Незавершенная”). Руководитель/система или текущий исполнитель (если не справляется и заполняет сложность). |
| Делегирована в другой отдел | Новый | Возврат в исходный отдел или принятие другим отделом. |
| В работе | Отклонена | Заявка отклонена. |
| В работе | Завершена | Все работы выполнены, исполнитель закрывает заявку. |
| В работе | Новый | Передача другому исполнителю (“Незавершенная”). Руководитель/система или текущий исполнитель (если не справляется и заполняет сложность). |
| Завершена | - | Заявка выполнена. |
| Отклонена | - | Заявка отклонена. |
